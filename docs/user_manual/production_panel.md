# Manual de Usuario: Panel de Producción (Gantt)

El Panel de Producción es la herramienta más potente para planificadores y supervisores. Utiliza un diagrama de Gantt interactivo para visualizar la carga de trabajo.

## Conceptos Básicos

*   **Eje Vertical (Y)**: Lista de Máquinas y Operadores.
*   **Eje Horizontal (X)**: Línea de tiempo (Días/Horas).
*   **Barras**: Representan una tarea de maquinado asignada a una máquina específica. El largo de la barra indica la duración.

## Acciones Principales

### 1. Visualizar Tareas
Cada barra muestra información básica (Nombre de la pieza, Cantidad).
*   **Hover**: Pasa el cursor sobre una barra para ver detalles (Fecha inicio/fin, Estatus completo).
*   **Línea de "Hoy"**: Una línea vertical roja marca el momento actual para ver qué debería estar pasando ahora mismo.

### 2. Editar Planeación
Tienes control total sobre la programación directamente en el gráfico:
*   **Mover (Reprogramar)**: Haz clic y arrastra una barra horizontalmente para cambiar su fecha u hora de inicio.
*   **Reasignar Máquina**: Arrastra una barra verticalmente para moverla de una máquina a otra.
*   **Redimensionar (Duración)**: Pasa el cursor por el borde derecho de una barra y arrástralo para extender o acortar la duración estimada del trabajo.
*   **Doble Clic**: Abre los detalles completos de la tarea para editar operador, notas, o estatus específicos.

### 3. Filtros y Búsqueda
Usa la barra de herramientas superior para filtrar el gráfico:
*   **Máquina**: Ver solo una máquina específica.
*   **Partida/Pieza**: Buscar por código de pieza para ver cuándo se fabricará.

### 4. Modo Pantalla Completa
Para reuniones o visualización en pantallas grandes en piso:
*   Haz clic en el icono de "Pantalla Completa" para ocultar la barra lateral y maximizar el área del gráfico.
