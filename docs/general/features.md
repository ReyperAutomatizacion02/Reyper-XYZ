# Funcionalidades del Sistema

## Módulos Principales

### 1. Control de Usuarios
*   **Registro y Autenticación**: Sistema de usuarios únicos con validación robusta de contraseñas.
*   **Login Social**: Opción de acceso mediante cuentas de Google.
*   **Gestión de Perfiles**: Roles y permisos diferenciados por departamento.

### 2. Dashboard Principal
*   **Indicadores Clave (KPIs)**: Gráficos de productividad y estado general de la planta.
*   **Navegación Intuitiva**: Barra lateral colapsable para maximizar el espacio de trabajo.
*   **Modo Oscuro/Claro**: Adaptabilidad visual según preferencia del usuario.

### 3. Planeación de Producción (Gantt Dinámico)
*   **Visualización**: Diagrama de Gantt interactivo para visualizar la carga de trabajo por máquina y operador.
*   **Edición en Tiempo Real**: Capacidad de redimensionar y mover tareas directamente en el gráfico.
*   **Filtros Avanzados**: Búsqueda por código de pieza, máquina o partida.
*   **Status Visual**: Diferenciación por colores para identificar rápidamente el estado de cada orden.

### 4. Gestión de Proyectos y Partidas (Sincronización Notion)
*   **Integración Bidireccional**: Los datos críticos (fechas, estatus) se sincronizan con bases de datos de Notion.
*   **Seguimiento de Piezas**: Trazabilidad desde "Solicitud" hasta "Entrega", incluyendo imágenes y especificaciones.

### 5. Ventas y Cotizaciones
*   **Cotizador Automático**: Generación rápida de cotizaciones formales.
*   **Historial**: Registro de todas las cotizaciones emitidas para consulta o duplicación.
*   **Cálculos Automáticos**: Totales, impuestos y subtotales calculados al instante.

### 6. Herramientas Auxiliares (Planeadas/En Desarrollo)
*   Solicitud de Materiales.
*   Solicitudes de Accesorios y Tornillería.
*   Estimador de Costos de Maquinado.
